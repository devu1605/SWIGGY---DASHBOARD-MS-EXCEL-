# SWIGGY---DASHBOARD-MS-EXCEL-

## 📌 Project Overview
This project is an interactive **MS Excel Dashboard** created to analyze and visualize Swiggy food delivery data. The dashboard converts raw data into meaningful insights using charts, pivot tables, slicers, and KPI metrics.

## 📊 Dashboard Insights
- Sales Performance Analysis
- Customer Ordering Trends
- Restaurant Performance Tracking
- Category-wise Analysis
- City-wise Order Distribution

## 🚀 Features
- Interactive Dashboard
- Dynamic Charts & Graphs
- Pivot Tables & Pivot Charts
- KPI Visualization
- Slicers and Filters
- Data Cleaning & Formatting

## 🛠️ Tools & Technologies
- Microsoft Excel
- Pivot Tables
- Conditional Formatting
- Data Visualization Techniques

## 🎯 Objective
The objective of this project is to transform raw Swiggy data into a professional dashboard that helps in better business analysis and decision-making.
